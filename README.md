> An electron-vue project

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


```


## 功能点
这个项目支持创建聊天对象，修改头像，发送各种类型的消息：文本、图片、语音、视频、语音通话、视频通话、转账与收账、文件、系统消息
